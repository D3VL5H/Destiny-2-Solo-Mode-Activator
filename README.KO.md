# 데스티니 가디언즈 솔로모드 활성기
가장 간단한 데스티니 가디언즈 솔로모드 활성기: D2Solo

# D2Solo는 어떤 동작을 하나요?
D2Solo는 방화벽 규칙을 만들어 데스티니 가디언즈에서 사용하는 포트를 막고,
이를 통해 다른 플레이어들과 매칭되지 않게 합니다.

# 어떻게 사용하나요?
간단합니다! 그냥 ON 버튼을 누르면 솔로모드가 켜질 것입니다.
